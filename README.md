## Run the app locally

### Clone the project

```bash
  git clone https://github.com/Ramandeep9898/btc-halving.git
```

### Go to the project directory

```bash
  cd btc-counter
```

### Install dependencies

```bash
  yarn
```

### Run the Development Server

```bash
  yarn dev
```
